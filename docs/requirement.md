前端系统设计作业
目标
实现一个视频素材管理平台的前端应用，重点考察组件设计能力。后端 API 通过 mock 自行实现（至少 20 条记录）。

允许使用的技术
框架选择 Vue 或 React 之一，并在文档中说明选择原因。

Part A · 核心页面实现
素材列表页
展示形式：以卡片或表格形式展示素材列表，支持分页。

FilterBar 组件：实现按状态、标签、上传人、城市进行过滤。

要求：过滤项通过配置数据驱动，不得硬编码。

SortControl 组件：支持按上传时间、文件大小多字段排序。

列表项内容：展示封面缩略图、标题、上传人、审核状态、文件大小。

素材详情页
交互：点击列表项进入详情页（或侧边抽屉）。

功能：展示素材完整信息与视频播放器。

字段裁剪：用户可勾选只显示部分字段。

附加目标：字段裁剪偏好持久化到 localStorage，刷新后保留。
